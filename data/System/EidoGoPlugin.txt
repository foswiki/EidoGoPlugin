---+ Eidogo TWiki Plugin
Visit http://eidogo.com for eidogo itself.

   1 install http://eidogo.com on twiki webserver (=svn co !http://eidogo.googlecode.com/svn/trunk/ eidogo=)
      * eg. local /var/www/myserver.com/htdocs/eidogo and !http://www.myserver.com/eidogo
   1 [[%TOPIC%#Plugin_Installation_Instructions][Install this Plugin]] 
   1 Upload your sgf files
      * eg. local /var/www/myserver.com/htdocs/sgf/test.sgf
   1 Add the Eidogo tags =%<nop>EIDOGO{game="/sgf/test.sgf" mode="eidogo-player-auto" title="testtitle"}%= or =%<nop>EIDOGO{"%ATTACHURL%/test.sgf"}%=

eventually change layout in !EidoGo in %ROOTDIR%/player/css/player.css in section =.eidogo-player.theme-compact=
  
---++ Testing Installation
%EIDOGO{game="%ATTACHURL%/test.sgf" mode="eidogo-player-auto" title="testtitle"}%

---++ Syntax Rules


   * Set DEBUG to 1 to get debug messages in =data/debug.txt=.  Default: =0=
      * Set DEBUG = 0

   * Default root directory for eidogo installation: =/eidogo/=
      * Set ROOTDIR = /eidogo/

   * Default mode for display: =eidogo-player-auto=, =eidogo-player-problem=
      * Set DEFAULT_MODE = eidogo-player-auto

   * Default mode for display: =eidogo-player-auto=, =eidogo-player-problem=
      * Set CONFIG = theme: "compact", showComments: true, showPlayerInfo: true, showGameInfo: true, showTools: false, showOptions: false, markCurrent: true, markVariations: true, markNext: false, problemMode: false, enableShortcuts: true

---++ Plugin Installation Instructions

   1 Unpack files in TWiki installation dir =lib/TWiki/Plugins/EidoGoPlugin.pm= and =data/TWiki/EidoGoPlugin.txt= =data/TWiki/EidoGoPlugin.txt,v= =%ATTACHURL%/test.sgf=
